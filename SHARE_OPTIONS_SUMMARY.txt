═══════════════════════════════════════════════════════════
  🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════

Problem: "Can't access the URL"
→ Make sure server is running
→ For ngrok: keep terminal open
→ For deployed: wait 2-3 minutes for build

Problem: "Excel file not working"
→ Run: pip install openpyxl
→ Restart server

Problem: "Port already in use"
→ Change port in main.py to 8000
→ Update ngrok command: ngrok http 8000
